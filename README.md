# 残灯巡夜

一个静态 Canvas 可玩原型，按设计案实现了 PC 俯视角即时动作肉鸽的 MVP。

## 运行

直接在浏览器打开 `index.html` 即可。

## GitHub Pages 上线

这是一个纯静态 HTML5 游戏，仓库根目录已经包含 `index.html`，可直接用 GitHub Pages 发布。

推荐步骤：

1. 在 GitHub 新建仓库，例如 `candeng-xunye`。
2. 将本项目文件推送到仓库默认分支。
3. 打开仓库 `Settings` → `Pages`。
4. `Build and deployment` 选择 `Deploy from a branch`。
5. `Branch` 选择 `main`，目录选择 `/root`。
6. 保存后等待 GitHub 生成访问地址。

## 操作

- `WASD` / 方向键：移动
- 鼠标：瞄准
- 自动发射镇魂符，鼠标左键可立刻出手
- `Space`：闪避

## 已实现内容

- 1 名角色：沈烬
- 1 张地图：长明旧街
- 10 个随机房间组成一局，Boss 固定压轴
- 普通战、精英战、怪谈事件、商人、Boss 前补给、Boss 战
- 6 个法器：朱砂符匣、引雷铜铃、照妖镜、魂灯、斩祟木剑、纸人替身
- 法器联动：
  - 朱砂符匣 + 引雷铜铃：符咒命中概率挂引雷
  - 魂灯 + 纸人替身：替身触发时引爆魂火
  - 照妖镜 + 斩祟木剑：镜影消散时释放环形剑气
- 3 种普通敌人：灯蛾妖、纸伞鬼、铜锣尸
- 精英词缀：急行、厚皮、乱响
- Boss：无面灯婆，含召唤、扇形灯火、熄灯暗区和三阶段节奏
- 横向解锁记录：最远房间、破夜次数、见过的法器
- 古风设定案卷：主页点击“巡夜司案卷”可查看人物与妖祟设定图
- 司库商店：对局结束会把剩余银钱带出为库银，可购买永久生效的强力道具
- 多角色系统：主页可选择不同巡夜人，每名角色拥有独立设定、属性和被动技能

## 司库商店道具

- 天师朱批：镇魂符基础伤害提升
- 夜行护符：每局最大生命提升
- 玄雷司铃：每第 9 次出手触发额外落雷
- 百鬼钱囊：每局开局额外携带银钱
- 巡夜金令：每局开局自带一枚令火

## 巡夜人

- 沈烬：初始可用。均衡型新人术士，技能“巡夜令”会更稳定地产生令火。
- 林青梧：累计带出 120 库银，或在一局中发现魂灯后解锁。擅长魂火回复。
- 裴玄策：清理任意精英房后解锁。每第 5 次出手触发小雷，雷击更强。
- 晏纸鸢：纸人替身救命 1 次后解锁。闪避留下纸影，纸影消散会放出小剑气。
- 唐灯客：击败无面灯婆 1 次后解锁。开局银钱更多，夜市商店价格降低。
